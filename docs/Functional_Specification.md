# Automated Conveyor Sorting System

## Purpose

Simulated Beckhoff TwinCAT 3 conveyor sorting system.

The system detects parts,
classifies good/reject parts,
tracks production,
handles faults,
and provides operator visualization.

## Features

- Start / Stop Control
- Production Tracking
- Reject Pusher
- Alarm Management
- Fault Handling
- Speed Control
- EtherCAT Architecture
- TwinSAFE Concepts
- Operator Visualization

## Hardware Architecture

Simulated Beckhoff CX Controller

EL1008 Inputs
EL2008 Outputs
EL4002 Analog Outputs
EL1904 TwinSAFE Inputs
EL2904 TwinSAFE Outputs
